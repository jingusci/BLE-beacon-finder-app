# v0.1.2

## Cambios respecto a v0.1.1
- La app ahora informa por audio la baliza conocida detectada.
- Se agregaron avisos por audio para `Cocina`, `Pieza`, `Living` y para el caso `sin balizas`.
- La reproduccion de audio quedo integrada al flujo principal de deteccion en `MainActivity`.
- Se agregaron botones de prueba manual en `BeaconScannerActivity` para verificar los audios sin depender del escaneo BLE.
- Los audios usados por la app quedaron centralizados en `app/src/main/res/raw/`.
- APK debug recompilado manteniendo la version `v0.1.2`.
